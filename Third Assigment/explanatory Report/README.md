# Explanatory Report

Αυτός ο φάκελος περιέχει το δεύτερο, επεξηγηματικό report για την 3η εργασία.

- `explanatory_report_Aa_logistic_regression.tex`: το report για το Ερώτημα Α, υποερωτήματα (a)-(c), δηλαδή Logistic Regression, Linear Discriminant Analysis και Gaussian Naive Bayes με μία προβλεπτική μεταβλητή κάθε φορά.
- `logistic_regression_Aa_results.csv`: ο συγκεντρωτικός πίνακας αποτελεσμάτων που χρησιμοποιείται στο report.
- `lda_Ab_results.csv`: ο συγκεντρωτικός πίνακας αποτελεσμάτων για τη Linear Discriminant Analysis.
- `gnb_Ac_results.csv`: ο συγκεντρωτικός πίνακας αποτελεσμάτων για το Gaussian Naive Bayes.

Το report μεταγλωττίζεται με:

```bash
xelatex explanatory_report_Aa_logistic_regression.tex
```
